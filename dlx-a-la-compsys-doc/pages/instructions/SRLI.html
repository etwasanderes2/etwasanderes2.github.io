SRLI

Shift Right Logical Immediate

Tags

  • format:immediate

  • category:logic:shift

  • category:logic

Syntax

SRLI register register immediate

Operation

Rd = Rs1 >>> (Imm % 32)

Beschreibung

Schiebt die Bit der Zahl in Rs1 um Imm (modulo 32) Stellen nach rechts. Es wird mit 0 aufgefüllt. Das Ergebnis wird in Rd gespeichert.

Da um Imm modulo 32 verschoben wird, entspricht ein Rechts Shift um 32 einem Rechts Shift um 0. Eignet sich nicht zur Division, hierzu siehe SRA.

Typische Verwendungszwecke

  • Verschieben von Bit
