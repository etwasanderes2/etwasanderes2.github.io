SLT

Set if Less Than

Tags

  • format:register

  • category:arithmetic:comparison

  • category:arithmetic

Syntax

SLT register register register

Operation

Rd = if (Rs1 < Rs2) then (1) else (0)

Beschreibung

Setzt Rd auf 1, wenn Rs1 echt kleiner Rs2 ist. An sonst wird Rd auf 0 gesetzt. Für echt größer einfach die Argumente umdrehen.

Typische Verwendungszwecke

  • Größer (>) Tests
  • Vergleiche
  • Verzweigungen
  • Schleifen
  • Coole Dinge wenn man Rd als Zahl statt als Logik-Wert betrachtet
