SUB

Subtract

Tags

  • format:register

  • category:arithmetic:math

  • category:arithmetic

Syntax

SUB register register register

Operation

Rd = Rs1 - Rs2

Beschreibung

Subtrahiert die Zahl in Rs2 von der in Rs1. Das Ergebnis wird in Rd gespeichert.

Typische Verwendungszwecke

  • Mathe
  • Adressberechnung
  • Negieren von Registern (mittels Rs1 = R0)
  • Kopieren von Registern (mittels Rs2 = R0)
