XORI

Exklusiv Oder Immediate

Tags

  • format:immediate

  • category:logic:bitwise

  • category:logic

Syntax

XORI register register immediate

Operation

Rd = Rs1 ^ Imm

Beschreibung

Wendet die XOR Operation auf die Zahl aus Rs1 und die Zahl aus dem Immediate bitweise an. Das Ergebnis wird in Rd gespeichert.

Das Immediate wird signed-extendet, also sind bei negativen Zahlen die oberen 16 bit alle 1

Typische Verwendungszwecke

  • Logik
  • "Flippen" von bit
  • Bitweises NOT (mittels #-1)
  • Initialisieren von Registern (mittels R0)
  • Kopieren von Registern (mittels #0)
