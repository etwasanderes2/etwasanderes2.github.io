Syntax

ADD register register register

Operation

Rd = Rs1 + Rs2

Beschreibung

Addiert die Zahlen in Rs1 und Rs2. Das Ergebnis wird in Rd gespeichert.

Typische Verwendungszwecke

  • Mathe
  • Adressberechnung
  • Kopieren von Registern (mittels R0)
