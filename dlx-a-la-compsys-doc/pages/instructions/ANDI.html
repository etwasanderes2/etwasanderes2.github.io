ANDI

And Immediate

Tags

  • format:immediate

  • category:logic:bitwise

  • category:logic

Syntax

ANDI register register immediate

Operation

Rd = Rs1 & Imm

Beschreibung

Ver-UND-et die Zahl aus Rs1 und die Zahl aus dem Immediate. Das Ergebnis wird in Rd gespeichert.

Das Immediate wird signed-extendet, also sind bei negativen Zahlen die oberen 16 bit alle 1

Typische Verwendungszwecke

  • Logik
  • Maskierung
  • Setzten von bit auf 0
