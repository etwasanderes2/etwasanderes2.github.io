SEQI

Set if Equal to Immediate

Tags

  • format:immediate

  • category:arithmetic:comparison

  • category:arithmetic

Syntax

SEQI register register immediate

Operation

Rd = if (Rs1 == Imm) then (1) else (0)

Beschreibung

Setzt Rd auf 1, wenn Rs1 gleich Imm ist. An sonst wird Rd auf 0 gesetzt.

Typische Verwendungszwecke

  • Vergleiche
  • Verzweigungen
  • Schleifen
  • Coole Dinge wenn man Rd als Zahl statt als Logik-Wert betrachtet
