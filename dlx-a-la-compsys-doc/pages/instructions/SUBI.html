SUBI

Subtract Immediate

Tags

  • format:immediate

  • category:arithmetic:math

  • category:arithmetic

Syntax

SUBI register register immediate

Operation

Rd = Rs1 - Imm

Beschreibung

Subtrahiert die Zahl im Immediate von der in Rs1. Das Ergebnis wird in Rd gespeichert. Um Immediate minus Register zu rechnen, negiere das Resultat dieses Befehls.

Typische Verwendungszwecke

  • Mathe
  • Adressberechnung
  • Kopieren von Registern (mittels #0)
  • Initialisieren von Registern auf bis zu 32768
