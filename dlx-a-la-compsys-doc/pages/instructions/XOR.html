XOR

Exklusiv Oder

Tags

  • format:register

  • category:logic:bitwise

  • category:logic

Syntax

XOR register register register

Operation

Rd = Rs1 ^ Rs2

Beschreibung

Wendet die XOR Operation auf die Zahlen in Rs1 und Rs2 bitweise an. Das Ergebnis wird in Rd gespeichert.

Typische Verwendungszwecke

  • Logik
  • "Flippen" von bit
  • Tausch zweier Registerwerte ohne Hilfsregister
  • Kopieren von Registern (mittels R0)
