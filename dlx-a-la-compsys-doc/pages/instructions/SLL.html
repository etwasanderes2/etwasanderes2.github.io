SLL

Shift Left Logical

Tags

  • format:register

  • category:logic:shift

  • category:logic

Syntax

SLL register register register

Operation

Rd = Rs1 << (Rs2 % 32)

Beschreibung

Schiebt die Bit der Zahl in Rs1 um Rs2 (modulo 32) Stellen nach links. Es wird mit 0 aufgefüllt. Das Ergebnis wird in Rd gespeichert. Ein Arithmetischer Links Shift existiert nicht, da er das Gleiche wäre wie der Logische.

Da um Rs2 modulo 32 verschoben wird, entspricht ein Links Shift um 32 einem Links Shift um 0.

Typische Verwendungszwecke

  • Verschieben von Bit
  • Multiplikation mit 2^Rs2
