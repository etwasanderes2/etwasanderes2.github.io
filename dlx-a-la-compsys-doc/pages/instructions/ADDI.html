ADDI

Add Immediate

Tags

  • format:immediate

  • category:arithmetic:math

  • category:arithmetic

Syntax

ADDI register register immediate

Operation

Rd = Rs1 + Imm

Beschreibung

Addiert die Zahl aus Rs1 und die Zahl aus dem Immediate. Das Ergebnis wird in Rd gespeichert.

Typische Verwendungszwecke

  • Mathe
  • Adressberechnung
  • Initialisieren von Registern (mittels R0)
  • Kopieren von Registern (mittels #0)
